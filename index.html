装修学习

发表于 Dec 6, 2018

施工保护

  • 张贴告示
    • 告示中的内容应包括：
      • 开工日期
      • 施工单位及联系人电话
  • 电梯保护
  • 公共地面保护
    • 第一层PU防潮布（防水、防渗）对毛坯作用不大，后期可能有用
    • 第二层 白板（防撞、缓冲）
    • 第三层 夹板 （加强）
  • 大门保护，这里肯定要特别注意，拿房应该是已经装好门了

卫生间装修

  • 卫生间若有位置移动或做垫高时应注意垫高用的材料，不能用废料；以防后期出现渗水等情况。
  • 浴室瓷砖美丑于打底也有一定的关系，打底要有一定的平整性。

阳台装修

  • 阳台也需要做防水
  • 如果需要考虑覆土的问题，应该如何处理。

装修材料等

  • 封板

树莓派初探

发表于 Aug 8, 2018

树莓派的初始登录

  • 登录名：pi
  • 登录密码：raspberry

树莓派基本操作

  1. 进入root模式
  • Ctrl+Alt+F6
  • Ctrl+Alt+F1
  1. sudo raspi-config 设置raspberry，类似于Windows开机时的Tab或者F12

VNC的安装

  1. 安装VNC应使用命令行，在安装软件包之前，应确保apt缓存是最新的，通过键入命令：sudo apt-get upadte

  2. 键入sudo apt-get install tightvncserver完成安装。

  3. 完成安装后，需要设置VNC的密码，在命令行输入：vncpassword然后输入密码。

  4. 另外当询问是否设置一个只能查看桌面（view-only）密码时，可以不用设置。

  5. 设置开启启动，输入命令sudo nano /etc/init.d/tightvncserver 在文件中粘贴如下内容

  6. 完成后，在控制台输入vncserver启动vnc服务，此时系统会为vnc服务分配一个数字（port），在通过其他设备连接时需要。

  7. 完成如下操作后，可以在电脑上安装VNC Viewer(官网下载)，点击File -> New connection，弹出Properties，在VNC Server中输入树莓派对应IP地址以及服务号，Name中可以输入树莓派名字也可以不进行输入，点击OK。点击File -> New connection，弹出Properties，在VNC Server中输入树莓派对应IP地址以及服务号，Name中可以输入树莓派名字也可以不进行输入，点击OK。

  8. 双击图标连接，点击continue，输入在树莓派上设置好的vnc密码，完成登陆。

  9. 另外手机或者pad操作同样可行（方法同电脑端基本一致）

因装vnc有问题，重启登入不了桌面系统

  1. 解决办法：改为无密码登录
  • 进入root，见树莓派基本操作
  • 按下列步骤完成
cd /               #回到根目录
rm -r /tmp         #删除tmp文件夹
mkdir /tmp          #新建文件夹
rm /home/用户名/.Xauthority #新建文件
